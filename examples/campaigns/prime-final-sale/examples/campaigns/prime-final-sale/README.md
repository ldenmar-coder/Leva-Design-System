# Prime Final Sale Campaign

## Purpose

This is an example of a strong Leva Sleep promotional campaign email for a final-day sale.

## Why this example works

- Clear urgency from the top bar: sale ends tonight
- Strong hero hierarchy with a bold discount message
- Simple promo code visibility
- Product cards are easy to scan
- Strong founder-style reassurance section
- Clear financing module
- In-store CTA included
- Consistent brand colours, spacing, buttons, and footer
- Good balance of urgency, product detail, reassurance, and trust-building

## Use as reference for

- Final-day sale campaigns
- Urgency-led promotional emails
- Product-grid campaign layouts
- Financing-led CTAs
- Founder-note sections
- Sale reminder emails
