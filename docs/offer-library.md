# Leva Sleep Offer Library

Recurring offer types and their positioning. **Promo codes are event-specific — always confirm the active code and percentage before any build** (see Email Build Standards, section 1). The codes below are recorded as references to past events, not standing codes.

---

## Demo Days Mega Sale

**Positioning:** Showroom models, discontinued inventory, and limited-run packages.

**Key messaging:**
- Limited inventory
- While quantities last
- Demo Days pricing
- Additional discount with the confirmed event code *(past event used `CLEAROUT10`, 10% off)*

---

## 72 Hour Sale

**Positioning:** Short-term urgency promotion.

**Key messaging:**
- Ends soon
- Limited time
- Act before pricing expires

---

## Financing Offer

**Key messaging:**
- Monthly payment first
- Zero down
- Affordable path to adjustable sleep

**Formatting:** Always lead with the monthly payment before the total price.
