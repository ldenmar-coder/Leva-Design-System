# Leva Design System

## Purpose

The Leva Design System is the official source of truth for Leva Sleep email design and development.

This repository contains:

- Brand guidelines
- HTML email standards
- Reusable email components
- Claude prompt library
- Email templates
- Product documentation
- Localization guidelines
- QA checklists
- Accessibility standards
- Rendering fixes

Its purpose is to ensure every email created for Leva Sleep is consistent, production-ready, accessible, and aligned with brand standards while supporting AI-assisted workflows.
